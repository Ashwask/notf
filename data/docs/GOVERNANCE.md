# NOTF Network Governance

## Principles

1. **Distributed, Not Centralized**
2. **Alignment, Not Uniformity**
3. **Trust-Based Invitation**
4. **Rotating Co-Stewardship**

## Roles

### Nudge Unit
- First point of contact
- Facilitates connections
- Maintains this repository
- Organizes convenings

### Members
- Contribute asks and offers
- Share resources
- Participate in calls
- Support each other

### Co-Steering (Emerging)
- Trusted partners
- Help guide direction
- Uphold transparency
- Support documentation

## Decision Making

- Decisions made transparently
- Discussion on monthly calls
- Documented in this repository
- Consensus-seeking approach

## Membership

- By invitation based on trust
- Must demonstrate alignment with principles
- Active participation expected
- Can transition to inactive status if needed
